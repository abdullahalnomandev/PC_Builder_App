## How to run the the project?


First, install node_modules :

```bash

yarn install
# or
npm install

```


Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
